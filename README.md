# Akasztófa Játék

<p align="justify">
Rendszeresen jártam a MondoCon-ra az egyik kis programjuk a Reggeli/Esti Lógás volt, ahol ugye az Akasztófa nevű játékot játszották a nézőkkel. És mikor láttam, hogy Paint programot használnak az jutott az eszembe, hogy le is programozhatnék oda valamit. Hozzá is kezdtem 2025 nyarán és végül 2025-ben az ősszel megrendezett MondoCon-on már élesben is futhatott.
</p>
